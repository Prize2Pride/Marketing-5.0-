# Prize2Pride — Digital Marketing 5.0 Encyclopedia Platform

**Powered by CodinCloud | Authored by Coach Roued El Fadhel**

## Overview

The Prize2Pride Encyclopedia Platform is a God-tier, multilingual digital marketing education system built with a modern full-stack architecture. It delivers a structured, encyclopedia-style course on Digital Marketing 5.0 — from absolute beginner to ASI-scale autonomous commerce mastery.

## Stack

- **Frontend:** React 19 + Tailwind CSS 4 + tRPC client
- **Backend:** Express 4 + tRPC 11 + Drizzle ORM
- **Database:** MySQL (TiDB)
- **AI:** Built-in LLM (GPT-4o-mini) for Coach Roued AI Avatar
- **Auth:** Manus OAuth
- **i18n:** Custom LanguageContext (EN / FR / AR-TN Darja + RTL)
