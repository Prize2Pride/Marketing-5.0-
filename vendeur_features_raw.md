# Vendeur.site Feature Extraction & Analysis (Red Team Report)

## 1. Platform Core Capabilities
- **AI Site Builder:** Conversational AI (Claude Opus 4.5) that builds and edits websites via voice/text commands. Includes live preview and Git-like versioning for easy rollbacks.
- **Unified Multi-channel Inbox:** Consolidates Messenger, Instagram, WhatsApp, and Facebook Marketplace. Claims 3x faster response times.
- **360° Business Management:** Integrated management of products, stock, orders, customers, employees, and couriers in real-time.
- **Logistics (TunisiaShip):** Native integration for label printing, real-time tracking, and return management.
- **Analytics & RFM:** Advanced customer segmentation (Recency, Frequency, Monetary) with infinite drill-down and CSV exports.
- **TikTok Integration (Upcoming):** Will include Reels publishing, AI hashtags, TikTok Ads, TikTok Shop, and unified DM/comment inbox.

## 2. Pricing & Scalability Potential
- **Starter (Free):** 1 site, 50 products, 1 physical store. Targeted at small/micro-merchants.
- **Pro (49 TND/month):** Unlimited sites/products, Claude Opus 4.5 AI, full multi-channel inbox, TunisiaShip, RFM analytics. Targeted at growing merchants and dropshippers.
- **Business (Custom):** Multi-vendor support, unlimited employees, API & webhooks. Targeted at large-scale enterprises and marketplaces.

## 3. Technical Stack & Integrations
- **AI:** Powered by Anthropic's Claude Opus 4.5.
- **Payments:** Local Tunisian integration (Konnect).
- **Social:** Meta Cloud API (Messenger, Instagram, WhatsApp).
- **Logistics:** TunisiaShip.
- **Platform Type:** Mobile-first PWA (Progressive Web App).

## 4. Strategic Potential for Merchants & Dropshippers
- **Augmented Merchants:** Ability to scale operations without increasing headcount by leveraging AI for site management and customer service.
- **Augmented Dropshippers:** Rapid testing of products and niches using the AI site builder and unified inbox to manage multiple storefronts from one place.
- **Enterprises:** API and webhook support allow for deep integration with existing ERPs or custom workflows.
